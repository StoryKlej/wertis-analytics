# wertis-analytics
Prywatne narzędzie analityki sprzedaży Allegro (tylko odczyt)
## O aplikacji

WERTIS Analytics to prywatne, lokalne narzędzie analityczne wykorzystywane
wyłącznie przez sklep WERTIS do analizy własnej sprzedaży na Allegro.

- Działa lokalnie, wyłącznie w trybie tylko-do-odczytu (read-only).
- Łączy się z Allegro REST API jedynie w celu pobierania danych na potrzeby własnych analiz.
- Nie modyfikuje żadnych danych w serwisie Allegro.
